The following website hosts a code quiz. 

You press the 'Start Quiz' button to begin a timed four question quiz that outputs an integer score based on how many questions you got correct. The high score is recorded and at the end of the game you are prompted to submit your intiials in a box. After pressing submit, the page displays a stored list of all the high scores and the intiials corresponding to the score for each attempt at taking the quiz. 

Alternatively, you can also click on the 'View Highscores' hyperlink on the top left corner of the screen if you would like to go straight to the high score page.
